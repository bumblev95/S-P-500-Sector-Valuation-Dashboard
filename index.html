S&P 500 섹터 밸류에이션 대시보드

Filter the S&P 500 by GICS sector, enter forward EPS and growth assumptions, then compare each company against its own scenario fair value and sector valuation context. Built as one static HTML file for GitHub Pages.

데이터 소스 상태
기본 목록을 불러오는 중입니다. “S&P 500 목록 새로고침”을 누르면 DataHub/GitHub에서 최신 구성종목 CSV를 가져옵니다.
가격은 먼저 내장된 가족용 FMP API 키를 사용하고, 실패하면 Yahoo/Stooq을 백업으로 시도합니다. 필요하면 이 브라우저에서만 키를 바꿔 사용할 수 있습니다.
